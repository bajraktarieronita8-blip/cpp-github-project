# C++ GitHub Collaboration Project

## Përshkrimi
Ky projekt është zhvilluar në C++ dhe demonstron menaxhimin e kodit burimor
duke përdorur GitHub, përfshirë versionimin e kodit, commit-et dhe dokumentimin
e projektit.

## Si të ekzekutohet
1. Klono repository-n:
   git clone https://github.com/bajraktarieronita8-blip/cpp-github-project.git
2. Kompilo kodin:
   g++ src/main.cpp -o app
3. Ekzekuto programin:
   ./app

## Menaxhimi i Projektit
- Projekti është zhvilluar individualisht
- Janë përdorur commit-e të qarta dhe të strukturuara
- Repository përmban dokumentim dhe strukturë të rregullt
